```go
package main
 
import (
  "fmt"
)
 
func main() {
  var s = []string{"1", "2", "3"}
  modifySlice(s)
  fmt.Println(s)
}
 
func modifySlice(i []string) {
  i[0] = "3"
  i = append(i, "4")
  i[1] = "5"
  i = append(i, "6")
}

```
Ответ:
```
Вывод: [3, 2, 3]
Слайс - это структура, которая содержит следующие поля: указатель на начало массива, len и cap. 
При передаче в функцию, можно менять уже существующие элементы внутри слайса, так как меняются они в массиве, но если cap будет переполнена,то будет создан массив с большим заначением cap и изменение уже будет происходить в новом массиве.
```
